# PREP-KC Student Insight Capture Sheet

*Teacher-facing note pages for turning class discussion into usable insight.*

Use one section per class period. Capture what students actually said, not only what you expected them to say.

---

## Why this sheet matters

The class is not only producing a presentation; it is also generating student insight PREP-KC can use. Use this sheet to track what pathways felt strong, what barriers felt real, what supports mattered, and what language resonated with students. Save both patterns and surprises.
